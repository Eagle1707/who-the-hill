What is Who The Hill?

Shazam, but for House members faces.

— jennifer steinhauer (@jestei) January 11, 2017

Who The Hill is a NYT internal facial recognition service for members of Congress.

Text a picture of a member of Congress (or a group of members) to {{ number }}. You'll receive a message back listing all the members of Congress in the picture. The message will also tell you how confident the application is about its judgement. In addition, you'll receive a message containing a copy of the picture you sent with colored outlines around the faces of each Congress member.

Please do not share the Who The Hill number outside of the Times. Every Who The Hill text costs a (small) amount of money, so we don't want random people using it.

Also, please note that while we hope Who The Hill is useful, we cannot guarantee 100 percent accuracy and occasionally see recognition errors. Please double check any result from Who The Hill before using it.

How does it work?

This app uses Amazon Rekognition, a facial recognition service that can analyze pictures and detect "celebrity" faces in them.

Amazon's facial recognition API has more trouble with some members of Congress than others, so this isn't a perfectly reliable solution. This means that the API may be unable to identify some Congress members and return a "failure to find a Congress member" message. We're working on ways to improve the recognition accuracy.

The texting function of Who The Hill is powered by Twilio, a service that allows software to send text messages.

If you're curious about the exact code behind Who The Hill, you can check out the Github repository.
